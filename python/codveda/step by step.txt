Step 2: Define business problem features

Weâ€™ll cluster posts based on engagement (Likes, Retweets) and optionally Sentiment.

# Select numeric features relevant for clustering
features = df[["Likes","Retweets"]].copy()

# Scale data (K-Means is distance-based â†’ scaling needed)
scaler = StandardScaler()
scaled_features = scaler.fit_transform(features)

Step 3: Find optimal number of clusters (Elbow Method)
# Try different cluster numbers
wcss = []  # within-cluster sum of squares

for i in range(1, 11):
    kmeans = KMeans(n_clusters=i, random_state=42, n_init=10)
    kmeans.fit(scaled_features)
    wcss.append(kmeans.inertia_)

# Plot elbow curve
plt.plot(range(1,11), wcss, marker="o")
plt.title("Elbow Method for Optimal k")
plt.xlabel("Number of Clusters")
plt.ylabel("WCSS")
plt.show()


âœ… Choose the â€œelbow pointâ€ (where curve bends) as best k.

Step 4: Apply K-Means clustering
# Letâ€™s assume k=3 based on elbow method
kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
df["Cluster"] = kmeans.fit_predict(scaled_features)

# Inspect cluster assignments
print(df[["Likes","Retweets","Cluster"]].head())

Step 5: Visualize Clusters
